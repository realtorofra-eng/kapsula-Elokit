תוכנית השיעור

ספריית השיעורים

כל שיעור שבנית נשמר כאן, אצלך במחשב. אפשר לפתוח, להציג, להעתיק או להוריד.

חומרי גלם

קבצי המקור של המרצה, מסודרים לפי שיעור. בוחרים שיעור — והסוכנים בונים ממנו מערך מלא בקול שלך.

"סדרי" מעתיק (לא מזיז ולא מוחק) את הקבצים שמכילים "שיעור N" בשמם לתוך קבצי מקור/שיעור NN/. המקור נשאר שלם. עובד ב‑Chrome/Edge במחשב על כתובת האינטרנט.

תמונות

תמונות בסגנון שפת הנשמה — לשקופיות, לאינסטגרם, לכל מקום. כל תמונה נוצרת בפלטה ובתחושה של המותג.

המנוע חינמי לגמרי (Flux) — לא צריך מפתח. לפעמים לוקח כמה שניות.

כאן תופיע התמונה

📸 פוסט לאינסטגרם

משפט אחד שנוגע → פוסט מרובע ממותג, מוכן לאינסטגרם.

דוגמאות בקול שלך (אופציונלי — אבל עוצמתי)

הדביקי קטעים שכתבת בעצמך, או משפטים מתוך שיעורים שיצאו בדיוק נכון. המערכת תלמד מהם ותחקה את הניסוח שלך. כל דוגמה בשורה נפרדת.

הגדרות

פעם אחת. אחר כך אף פעם לא יוצאים מהמערכת.

📲 התקנת המערכת כאפליקציה

מתקינים פעם אחת — והמערכת נפתחת כאפליקציה אמיתית עם אייקון משלה, במסך מלא, בלי שורת כתובת. עובד במחשב (Windows · Linux · Mac) ובנייד (iPhone · Android).

מנוע הבינה

אם פרסת את המערכת ל‑Vercel והגדרת שם את המפתח — לא צריך כלום כאן. אם את מריצה מקומית — הזיני מפתח (נשמר רק אצלך במחשב).

console.anthropic.com → API Keys → Create Key.

לוגו

הלוגו מופיע בכותרת, במצגות, ב‑PDF ובכל תוצר. אפשר להעלות לוגו אחר (מומלץ PNG עם רקע שקוף).

עזרים חזותיים — וידאו ותמונות אמיתיים

המערכת תמיד יוצרת תמונות בינה ואייקונים (חינם, ללא מפתח). כדי שתחפש גם וידאו ותמונות אמיתיים מהרשת (למשל סרטון מים פועם למושג "מים חיים") — הוסיפי מפתח Pexels חינמי.

חינם ב‑pexels.com/api — הרשמה של דקה. בלי המפתח, השקופיות מקבלות תמונות בינה ואייקונים.

גיבוי מקומי

שומר/טוען את כל הספרייה, הקול והדוגמאות בקובץ אחד.

תיקיית העבודה — חיבור לקבצים שלך

חיבור התיקייה שבה את שומרת את קבצי השיעורים. כך, במקום להעלות מסמך-מסמך, את בוחרת מתוך התיקייה — ועוזרת ההוראה יכולה לקרוא ממנה ולשמור אליה תוצרים.

חיבור-קבוע נתמך ב‑Chrome/Edge במחשב, וכשמריצים את המערכת מכתובת אינטרנט (Vercel) או משרת מקומי — לא בפתיחה ישירה של הקובץ. בכל מצב אחר אפשר להשתמש ב‑"בחרי מתיקייה שלמה" בלשונית "שיעור חדש", שעובד תמיד.

סנכרון ענן — ספרייה בכל המכשירים

מאפשר לראות את אותה ספרייה בנייד ובמחשב. חינם דרך Supabase. הקמה (פעם אחת):

  1. נכנסים ל‑supabase.com → New Project (חינם).
  2. ב‑SQL Editor מריצים: create table sn_library (id text primary key, data jsonb); alter table sn_library disable row level security;
  3. Settings → API → מעתיקים Project URL ו‑anon key לכאן.
