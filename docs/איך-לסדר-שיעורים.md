# איך לסדר את השיעורים

לכל שיעור תיקייה אחת עם מספר. בתוכה שתי תת־תיקיות:

```
שיעור 01/
  גלם/     ← לכאן זורקים את כל הקבצים מהמורָה (PDF, וורד, סיכום, תמלול — כל מספר)
  תוצר/    ← לכאן נשמרת תוכנית השיעור המוכנה + המצגת
```

**כללים פשוטים:**
- ב"גלם" לא נוגעים. זה המקור, נשאר נקי.
- כל מה שהמערכת יוצרת הולך ל"תוצר".
- שיעור 01 כבר מוכן כדוגמה. לשיעור חדש — מעתיקים את התיקייה ומשנים מספר.

ההכנה הזו תואמת את הסטטוסים בלוח הבקרה: **גלם → בעבודה → מוכן**.
