# שפת הנשמה — המערכת שלך 🌿
### התחילי כאן

מערכת אחת, סגורה, שהופכת קבצים מהמורָה → לשיעור שנשמע **כמוך**, מציגה אותו עם הערות פרטיות,
ומייצרת תמונות בסגנון המותג. **הכול קורה בתוך המערכת.**

---

## הקובץ הראשי: `index.html`
דאבל־קליק → נפתח בדפדפן. זאת כל המערכת. (זה גם הקובץ שעולה ל‑Vercel.)

### הכנה — פעם אחת
המערכת צריכה "מנוע בינה". יש שתי דרכים:
- **מקומית (הכי מהיר להתחלה):** לשונית **הגדרות** → מדביקים מפתח מ‑[console.anthropic.com](https://console.anthropic.com/settings/keys) → שמירה. נשמר רק אצלך.
- **באוויר (מומלץ לטווח ארוך):** מעלים ל‑Vercel והמפתח נשמר בשרת — אז לא צריך להזין אותו בכלל. מדריך: [פריסה-ל-Vercel.md](פריסה-ל-Vercel.md).

## הלשוניות
| לשונית | מה היא עושה |
|---|---|
| **שיעור חדש** ⭐ | גוררים קבצים → 4 הסוכנים כותבים שיעור בקול שלך, חי על המסך |
| **ספרייה** | כל השיעורים שבנית — פתיחה, **הצגה**, העתקה, הורדה |
| **תמונות** | מחולל תמונות חינמי בסגנון המותג — לשקופיות ולאינסטגרם |
| **פרופיל הקול** | הלב. עורכים אותו + מוסיפים "דוגמאות בקול שלך" שהמערכת לומדת מהן |
| **הגדרות** | מפתח (לשימוש מקומי), איכות מנוע, גיבוי |

## מצב הצגה (מסך קהל נקי + הערות שרק את רואה)
בכל שיעור — כפתור **🖥️ הצג מצגת**:
- **המסך שלך:** שקופית נוכחית · **הערות דובר פרטיות** · השקופית הבאה · טיימר · חצים.
- **🔆 פתח מסך קהל:** נפתח חלון נקי — גוררים אותו למקרן, לוחצים "מסך מלא". התלמידות רואות
  רק את השקופית. הכול מסונכרן: כשאת מעבירה שקופית, מסך הקהל מתעדכן.
- חצים במקלדת מעבירים שקופיות. אפשר גם לייצא ל‑PowerPoint/Reveal.

---

## כל שיעור חדש (השגרה)
1. **שיעור חדש** → גוררת קבצים (PDF/וורד/תמונה/טקסט) → שם ומספר → בוחרת אורך.
2. **✨ בנה את השיעור** — רואה את השיעור נכתב חי.
3. לא מדויק? **🔁 תיקון בקול** ("תגעי יותר", "תקרקעי כאן").
4. **💾 שמירה** → **🖥️ הצג מצגת** → מלמדת.
